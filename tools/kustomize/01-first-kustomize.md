
# First Kustomize

### Install Kustomize

```bash
// arm
wget https://github.com/kubernetes-sigs/kustomize/releases/download/kustomize%2Fv4.5.4/kustomize_v4.5.4_linux_arm64.tar.gz

tar -zxvf kustomize_v4.5.4_linux_arm64.tar.gz

sudo cp ./kustomize /usr/bin/
```
